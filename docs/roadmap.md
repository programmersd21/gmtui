# Roadmap

- [x] Async event loop with channel-based input routing
- [x] Gmail client: list, get, send, delete, modify labels
- [x] Cache with TTL and lazy loading (load more)
- [x] Search overlay with live filtering
- [x] Email view with header + scrollable body
- [x] Composer with reply prefill and send
- [x] Sorting (date/sender/subject)
- [x] Sidebar with category unread counts
- [x] Theme system with dark/light presets
